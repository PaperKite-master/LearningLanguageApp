# PROJECT_CONTEXT.md

## Tổng quan dự án

Tên dự án: **Japanese Learning App for IT**

Đây là một nền tảng học **tiếng Nhật dành riêng cho sinh viên IT và lập trình viên**.

Mục tiêu của ứng dụng:

* Học từ vựng tiếng Nhật trong môi trường **IT / công ty công nghệ**
* Luyện **ngữ pháp sử dụng trong giao tiếp công việc**
* Luyện đọc hiểu **tài liệu kỹ thuật**
* Luyện typing bảng chữ cái Nhật
* Ôn tập bằng **quiz, flashcard và mini game**

---

# Công nghệ sử dụng

Frontend stack:

* Vite
* React
* JavaScript
* CSS / Tailwind (optional)

Project này **chỉ bao gồm Frontend**.
Backend API sẽ được tích hợp sau.

Thiết kế giao diện dựa trên **Figma design**.

---

# Đối tượng người dùng

Ứng dụng dành cho:

* Sinh viên ngành **Công nghệ thông tin**
* Lập trình viên muốn làm việc với **công ty Nhật**
* Người học tiếng Nhật với mục tiêu **IT / Software Development**

Ví dụ nội dung bài học:

* IT vocabulary
* Software development terms
* Workplace communication
* Technical reading
